Реализация упорядоченного множества на языке Haskell
====================================================

_Этот README составлен с использованием [Markdown] [md] — специального легковесного языка разметки.
Его можно читать в блокноте или преобразовать в HTML (например, с помощью [Dingus] [d])._

[md]: http://daringfireball.net/projects/markdown
[d]: http://daringfireball.net/projects/markdown/dingus

Данный пакет содержит модуль `OrderedList.hs`, в котором определён тип данных «упорядоченное множество»
и реализованы теоретико-множественные операции над ним (вместе со вставкой элемента во множество).
Все операции выполняются за линейное относительно размера множеств время.

Пакет снабжён тестами для QuickCheck, которые проверяют корректность работы на случайных тестах (за эталон
берутся операции из пакета Data.List). Они находятся в файле `Main.hs`. Чтобы запустить тесты, можно выполнить
команду `runhaskell Main.hs`.
